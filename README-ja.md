# PseudoRedPower-Core
PseudoRedPower-Coreは、MinecraftのMODであり、**RedPower2を1.8.9以降に移植したもの**だ。だから、もしあなたがこのMODを面白いと思ってくれたのなら、**RedPower2の作者を尊敬してほしい**。

## 特徴
* 壁に設置することが出来るレッドワイヤー
* 集積回路
* 分割ブロック
* 気送管による輸送
